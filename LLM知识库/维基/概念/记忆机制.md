# 记忆机制（Memory）

## 定义

记忆机制是 [[LLM知识库/维基/概念/AI Agent|AI Agent]] 存储、管理和利用信息的能力。由于 LLM 本身的上下文窗口有限，Agent 需要外部记忆系统来维持长期对话、积累经验、支撑 [[LLM知识库/维基/概念/任务规划|规划]] 和决策。

## 记忆分类

### 按时间跨度
- **短期记忆（Working Memory）**：当前对话上下文、最近交互，通常存在 LLM 的上下文窗口中
- **长期记忆（Long-term Memory）**：持久化存储的历史信息、用户偏好、知识库

### 按获取方式（个人 LLM Agent 视角）
- **日志记录（Logging）**：直接记录原始交互数据
- **推断记忆（Inference）**：从交互中提取高级知识（如用户偏好、行为模式）
- **自我进化（Self-evolution）**：Agent 主动反思和更新自身知识

### 按存储形式
- **向量记忆**：使用嵌入向量 + 向量数据库（FAISS、Milvus、ScaNN）
- **结构化记忆**：知识图谱、数据库表
- **文本记忆**：自然语言摘要、经验总结

## 记忆在 Agent 中的应用

### 记忆增强规划
在 [[LLM知识库/维基/概念/任务规划|规划]] 过程中检索相关记忆辅助决策：
- **RAG 式记忆**：检索增强生成，用相关知识增强规划上下文
- **具身记忆**：从物理环境交互中积累的操作经验
- 代表工作：REMEMBER、MemoryBank、MemGPT

### 多 Agent 中的记忆
- **共享记忆**：多个 Agent 共享的公共知识库
- **私有记忆**：每个 Agent 独立维护的经验
- Stanford 的 Generative Agents 项目展示了 Agent 通过记忆流（Memory Stream）进行社会模拟

### 个人 Agent 中的记忆
- 用户画像、偏好、习惯的持久化
- 隐私保护下的记忆管理（本地处理、数据脱敏）

## 关键挑战

- **检索效率**：大规模记忆中快速准确检索
- **记忆遗忘**：何时删除过时信息
- **隐私保护**：个人记忆的安全存储
- **一致性**：跨会话的记忆一致性维护
