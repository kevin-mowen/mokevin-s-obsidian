# 工具使用（Tool Use）

## 定义

工具使用是 [[LLM知识库/维基/概念/AI Agent|AI Agent]] 通过调用外部工具来扩展自身能力的机制。LLM 本身擅长推理和生成文本，但通过工具可以实现代码执行、网络搜索、数据库查询、API 调用等操作。

## 工具类型

| 类型 | 示例 | 用途 |
|------|------|------|
| 搜索引擎 | Google, Bing | 获取最新信息 |
| 代码解释器 | Python REPL | 计算、数据分析 |
| API 调用 | 天气、地图、支付 | 与外部服务交互 |
| 文件操作 | 读写文件 | 数据持久化 |
| UI 自动化 | 点击、输入、滑动 | 操控应用程序 |
| 感知工具 | 摄像头、传感器 | 获取环境信息 |

## 代表性工作

- **Toolformer**：LLM 自主学习何时及如何使用工具
- **HuggingGPT**：调度 HuggingFace 上的专业模型作为工具
- **ReAct**：交替进行推理（Reasoning）和行动（Action）
- **个人 Agent 中的 UI 自动化**：AppAgent、AutoDroid 等通过操作手机 UI 完成任务

## 与其他能力的关系

- 工具使用扩展了 [[LLM知识库/维基/概念/任务规划|规划]] 的行动空间
- 工具反馈支撑 [[LLM知识库/维基/概念/反思与优化|反思与优化]]
- 工具结果可以存入 [[LLM知识库/维基/概念/记忆机制|记忆]]
