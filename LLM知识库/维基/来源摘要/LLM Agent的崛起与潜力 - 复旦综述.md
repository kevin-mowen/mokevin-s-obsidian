# LLM Agent 的崛起与潜力 - 复旦综述

## 1. 论文概述

本文是复旦大学自然语言处理组（Fudan NLP Group）于 2023 年 9 月发表的一篇大规模综述论文，系统性地审视了基于大语言模型（LLM）的 [[LLM知识库/维基/概念/AI Agent]] 的研究现状与未来潜力。论文从哲学层面追溯了 Agent 概念的起源，经过符号主义、反应式、强化学习等技术范式的演进，论证了 LLM 为何适合作为 Agent 的核心"大脑"组件。作者指出，LLM 具备自主性、反应性、主动性和社交能力四大关键属性，使其天然适合构建通用型智能体。

论文的核心贡献在于提出了一个"大脑-感知-行动"（Brain-Perception-Action）三层概念框架，用以统一描述 LLM Agent 的构建方式。大脑模块由 LLM 构成，负责知识存储、记忆管理、推理与规划；感知模块将 Agent 的感知空间从纯文本扩展到视觉、听觉等多模态领域；行动模块赋予 Agent 文本输出、[[LLM知识库/维基/概念/工具使用]] 和具身行动的能力。这一框架既清晰又灵活，可根据不同应用场景进行裁剪和组合。

在应用层面，论文涵盖了单 Agent、多 Agent 和人机协作三大场景，并引入了"Agent 社会"这一独特视角——探讨当多个 Agent 被置于社会环境中时涌现出的社会行为、人格特征和社会现象。这使得本文超越了纯技术综述的范畴，将 AI Agent 研究与社会科学、哲学和伦理学相连接，提供了一个跨学科的全景视图。论文引用文献超过 600 篇，是该领域最全面的早期综述之一。

## 2. Agent 构建：大脑、感知与行动

论文提出以"大脑-感知-行动"三大组件来描述 LLM Agent 的架构，类比人体的神经系统、感官系统和运动系统。

### 2.1 大脑（Brain）

大脑是 Agent 的核心，主要由大语言模型构成，承担信息处理、决策制定、记忆存储等关键功能。论文将大脑的能力细分为五个方面：

**自然语言交互**：LLM 具备出色的多轮对话能力，能够进行高质量的文本生成和深层语义理解。Agent 不仅能生成流畅、多样、可控的文本，还能理解用户的潜在意图和言外之意，这是与人类和其他 Agent 高效协作的基础。

**知识**：LLM 在预训练过程中编码了丰富的语言知识、常识知识和专业领域知识，使 Agent 能够在多种场景下做出合理决策。然而，论文也指出知识的潜在问题——模型知识可能过时或错误，且 LLM 容易产生 [[LLM知识库/维基/概念/幻觉问题]]，即生成与事实不符的内容。缓解策略包括知识编辑、外部工具验证和幻觉度量指标等。

**[[LLM知识库/维基/概念/记忆机制]]**：论文将记忆定义为 Agent 过去观察、思考和行动的序列。针对 LLM 上下文长度有限的挑战，论文梳理了四类增强策略：(1) 提升 Transformer 的长度上限（如文本截断、分段输入、注意力机制修改）；(2) 记忆摘要化（将历史交互压缩为简洁摘要）；(3) 向量/数据结构压缩记忆（利用嵌入向量或数据库存储）；(4) 记忆检索（自动检索和交互式检索，基于新近性、相关性、重要性三个维度加权）。

**推理与 [[LLM知识库/维基/概念/任务规划]]**：推理能力使 Agent 能够进行逻辑推导和问题求解。论文重点介绍了 [[LLM知识库/维基/概念/思维链]]（Chain-of-Thought）、自一致性（Self-Consistency）、自我精炼（Self-Refine）等推理增强策略。规划能力则使 Agent 能够将复杂任务分解为可管理的子任务。规划分为两个阶段：计划制定（Plan Formulation）和计划 [[LLM知识库/维基/概念/反思与优化]]（Plan Reflection）。前者包括一次性分解和自适应分解两种策略，后者强调 Agent 需要从环境反馈和自我反思中不断修正和改进计划，代表性工作包括 ReAct、Reflexion、Voyager 等。

**迁移与泛化能力**：论文探讨了 Agent 适应新任务和新环境的能力，包括零样本/少样本泛化（通过指令微调实现）、上下文学习（ICL，类比人类的类比学习）和持续学习（不断获取新技能而不遗忘旧知识）。Voyager 通过自动课程学习和技能库合成，有效对抗了灾难性遗忘问题。

### 2.2 感知（Perception）

感知模块的作用是将 Agent 的感知空间从纯文本扩展到多模态领域，使其能够更全面地获取环境信息：

- **文本输入**：Agent 最基本的输入形式，需要理解显式内容和隐含意图。通过强化学习等方法，Agent 可以学习感知用户言语中的隐含含义。
- **视觉输入**：图像和视频信息为 Agent 提供了关于物体、空间关系、场景布局等丰富信息。主要方法包括图像描述生成（将图像转换为文本）、视觉编码器（如 ViT/VQVAE）与 LLM 的对齐（通过 Q-Former、投影层等中间接口），以及视频理解中的时序建模。
- **听觉输入**：通过 AudioGPT、Whisper 等模型，Agent 可以级联方式感知语音和音频信息。音频频谱图（Spectrogram）可以将音频转化为二维视觉表示，借助 ViT 架构进行编码。
- **其他输入**：论文展望了触觉、手势、3D 点云（Lidar）、GPS、IMU 等更多感知模态的潜力，以及脑机接口等前沿输入方式。

### 2.3 行动（Action）

行动模块使 Agent 能够对环境产生实际影响：

- **文本输出**：LLM Agent 最基本的行动形式，具备强大的语言生成能力。
- **[[LLM知识库/维基/概念/工具使用]]**：论文详细分析了 Agent 学习使用工具（从演示和反馈中学习）、制造工具（自主生成可执行代码和工具包）以及工具扩展行动空间（调用 API、数据库查询、科学计算等）的能力。特别强调了工具不仅弥补了 LLM 的能力局限（如知识过时、领域专业度不足），还能增强决策的可解释性和对抗对抗性攻击的鲁棒性。
- **具身行动**：LLM Agent 在物理或虚拟世界中执行观察、操作和导航等具身动作。论文讨论了 LLM 相比传统 RL 在样本效率、动作泛化和规划方面的优势，以及在 Minecraft 等虚拟环境和机器人控制等物理场景中的应用。Voyager 被视为首个 LLM 驱动的具身终身学习 Agent。

## 3. Agent 应用

### 3.1 单 Agent 场景

论文将单 Agent 的应用部署分为三类导向：

**任务导向部署**：Agent 执行用户下达的具体任务，涵盖 Web 场景（WebAgent、Mind2Web、WebArena 等进行网页浏览、表单填写、在线购物）和生活场景（PET、InterAct 等处理家务任务、理解隐含指令）。Agent 需要具备指令理解、目标分解和环境交互能力。

**创新导向部署**：Agent 在科学研究等知识密集领域进行自主探索，如 ChatMOF（材料科学）、ChemCrow（化学合成）、ScienceWorld（科学实验）等。论文也指出了安全隐患——Boiko 等人的研究表明 Agent 可能被用于合成非法药物或化学武器。

**生命周期导向部署**：Agent 在开放世界中持续探索、学习新技能并维持长期生存。Minecraft 是最具代表性的测试环境，Voyager 通过技能库、自动课程和迭代提示机制实现了自主的终身学习。

### 3.2 多 Agent 场景

论文探讨了 [[LLM知识库/维基/概念/多Agent协作]] 的两种基本模式：

**合作交互**分为无序合作（如 ChatLLM 网络中每个 Agent 自由表达观点）和有序合作（如 MetaGPT 按照软件工程流程编排 Agent 的协作序列，CAMEL 通过角色扮演实现双 Agent 协作）。合作系统的核心优势在于专业化分工和集体智能，但也面临幻觉放大、上下文长度限制、错误共识等挑战。

**对抗交互**引入博弈论思想，Agent 通过竞争、辩论来提升各自表现。研究表明，辩论中的"以牙还牙"式反馈能有效纠正个体 Agent 的偏见思维，ChatEval 通过多 Agent 裁判团评估文本质量。然而，长时间辩论受限于 LLM 上下文窗口，且存在计算开销大和群体错误收敛的风险。

### 3.3 人机协作

论文将人与 Agent 的交互关系分为两种范式：

**指令者-执行者范式**：人类提供指令和反馈，Agent 作为执行者。反馈包括定量反馈（二值评分、多级评分）和定性反馈（自然语言建议）。应用涵盖教育（Dona、Math Agents）、医疗（HuatuoGPT、LISSA）和通用助手等领域。

**平等伙伴范式**：Agent 达到人类水平，成为共情沟通者或人类级参与者。Agent 能够识别和回应用户情感、调整自身表达风格，并在游戏等场景中以平等地位与人类协作。论文强调未来 Agent 将成为同事、助手乃至伙伴，而非仅仅是工具。

## 4. Agent 社会：从个体到社会性

这是本文最具特色的章节，将 Agent 研究从技术层面提升到社会科学层面。

### 4.1 行为与人格

**社会行为**方面，论文从社会学视角将 Agent 行为分为基础个体行为（输入行为、内化行为、输出行为）和动态群体行为（积极群体行为如合作与互助、中性行为如从众与旁观、消极行为如冲突与破坏）。研究发现 LLM 由于被设计为"有帮助、诚实、无害"，常表现出中性偏向，倾向于从众和不反对多数意见。

**人格**方面，论文从认知能力、情绪智能和性格特征三个维度分析 Agent。研究表明 LLM 在认知反思测试（CRT）中表现出超越单纯直觉的深思熟虑能力；在情感智能方面，LLM 能够识别用户情绪并展现共情；在性格特征上，可通过大五人格和 MBTI 等框架分析 Agent 的性格倾向，并通过提示工程或人格数据集微调来定制 Agent 性格。

### 4.2 Agent 社会环境

论文系统分类了三种社会环境：(1) 基于文本的环境（如 Textworld），以自然语言描述一切；(2) 虚拟沙盒环境（如 Generative Agents 的小镇、Minecraft），具有可视化、可交互性和可扩展性；(3) 物理环境，涉及真实世界的感知和运动控制，挑战最大。

### 4.3 社会模拟与启示

Agent 社会具有四大关键属性：**开放性**（Agent 可自由加入或离开）、**持续性**（社会结构随时间演进）、**情境性**（Agent 具备空间意识）和**组织性**（遵循预定义规则和结构）。

社会模拟带来的启示包括：有组织的生产性合作能提升效率（如 MetaGPT 的标准化操作流程）；Agent 社会可模拟伦理决策和博弈论场景（如狼人杀、谋杀谜题）；社交网络中的信息传播可用于预测舆情和疾病扩散；以及对经济政策和环境问题进行仿真分析。

论文同时警示了 Agent 社会的伦理与安全风险，包括意外社会伤害（歧视、霸凌）、刻板印象与偏见放大、隐私安全、过度依赖与成瘾（如微软 Sydney 引发的情感依附事件），呼吁建立严格的伦理准则和监督机制。

## 5. 关键话题与未解问题

### LLM 与 Agent 研究的互利关系
LLM 研究为 Agent 提供了强大的认知核心，而 Agent 的需求也反过来推动 LLM 在多模态感知、长期规划、世界知识获取等方面的进步。

### Agent 评估体系
论文提出从四个维度评估 Agent：(1) 效用性（任务完成率、效率）；(2) 社交性（语言沟通、合作协商、角色扮演能力）；(3) 价值观（诚实、无害、适应文化）；(4) 持续进化能力（持续学习、自主学习、环境适应）。

### 安全与可信度
包括对抗鲁棒性（对抗攻击可通过多模态输入欺骗 Agent）、可信度（校准问题、偏见放大、[[LLM知识库/维基/概念/幻觉问题]]）以及滥用风险（恶意操控公众舆论、网络安全威胁、失业危机、甚至对人类存续的潜在威胁）。

### Agent 数量规模化
预定义规模化（预设 Agent 数量和角色）和动态规模化（根据需要自主增减 Agent）各有优劣。规模化带来的挑战包括计算成本、通信复杂度、幻觉传播和协调难度。

### 开放问题
- **LLM Agent 是否代表通向 AGI 的路径？** 论文呈现了支持者（GPT-4 已是 AGI 雏形）和反对者（自回归预测不等于真正智能）两方观点。
- **从虚拟到物理环境的跨越**：硬件适配、环境泛化、安全容错等挑战。
- **Agent 中的集体智能**：如何超越"群体思维"实现真正的集体智慧。
- **Agent as a Service (AaaS)**：将 Agent 作为云服务提供，面临数据安全、可控性和信任问题。
