# FastGPT 架构总览

> 版本：4.14.3.9 | 分析日期：2025-12-06

---

## 1. 项目简介

FastGPT 是一个功能完整的 **AI Agent 构建平台**，通过可视化工作流提供开箱即用的数据处理、模型调用能力。基于 NextJS 构建的全栈 TypeScript 应用，后端使用 MongoDB/PostgreSQL。

### 核心定位

- **低代码 AI 应用构建**：通过拖拽式工作流编排复杂 AI 应用
- **知识库增强**：支持多源数据导入和向量检索增强生成 (RAG)
- **多模型集成**：统一接口支持多种 AI 提供商
- **企业级特性**：多租户、权限管理、使用计费

## 3. Monorepo 架构

```
fastgpt/
├── packages/                    # 共享库代码
│   ├── global/                  # 全局类型、常量、工具函数
│   ├── service/                 # 后端服务、数据库模型、业务逻辑
│   └── web/                     # 前端共享组件、Hooks、状态管理
│
├── projects/                    # 应用程序
│   ├── app/                     # 主 NextJS 应用 (前端 + API)
│   ├── marketplace/             # 模板市场应用
│   ├── sandbox/                 # NestJS 代码执行沙箱
│   └── mcp_server/              # MCP 协议服务器
│
├── plugins/                     # 外部插件
│   ├── model/                   # AI 模型插件 (9个)
│   └── webcrawler/              # 网页爬虫插件
│
├── deploy/                      # 部署配置 (Docker/Helm)
├── document/                    # 文档站点
└── test/                        # 测试文件
```

## 5. 核心数据流

### 5.1 应用创建与编辑流程

```
┌─────────────────────────────────────────────────────────────────┐
│                         前端 (React)                            │
├─────────────────────────────────────────────────────────────────┤
│  AppList 页面                      AppDetail 页面               │
│  ├─ useRequest(getAppList)         ├─ ReactFlow 编辑器          │
│  └─ 创建/删除/复制应用              ├─ 节点拖拽配置               │
│                                    └─ 保存/发布版本              │
└────────────────────────┬────────────────────────────────────────┘
                         │ API 调用
┌────────────────────────▼────────────────────────────────────────┐
│                    API 层 (NextJS API Routes)                   │
├─────────────────────────────────────────────────────────────────┤
│  /api/core/app/create              /api/core/app/update         │
│  /api/core/app/list                /api/core/app/version/publish│
│  /api/core/app/del                 /api/core/app/detail         │
└────────────────────────┬────────────────────────────────────────┘
                         │ 调用 Service
┌────────────────────────▼────────────────────────────────────────┐
│                   Service 层 (packages/service)                 │
├─────────────────────────────────────────────────────────────────┤
│  core/app/controller.ts            core/app/version/controller  │
│  ├─ beforeUpdateAppFormat()        ├─ createAppVersion()        │
│  ├─ findAppAndAllChildren()        └─ publishAppVersion()       │
│  └─ onDelOneApp()                                               │
└────────────────────────┬────────────────────────────────────────┘
                         │ 数据库操作
┌────────────────────────▼────────────────────────────────────────┐
│                    数据层 (MongoDB)                             │
├─────────────────────────────────────────────────────────────────┤
│  MongoApp (apps 集合)              MongoAppVersion (versions)   │
│  ├─ _id, name, type                ├─ appId, nodes, edges       │
│  ├─ modules[], edges[]             ├─ isPublish, versionName    │
│  └─ chatConfig                     └─ time, tmbId               │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 对话执行流程

```
用户发送消息
      │
      ▼
┌─────────────────┐
│  前端 Chat 组件  │
│  useChatStore   │
└────────┬────────┘
         │ POST /api/v1/chat/completions
         ▼
┌─────────────────────────────────────────┐
│           API 权限认证                   │
│  ├─ Token 认证 (用户会话)               │
│  ├─ API Key 认证 (第三方)               │
│  └─ 分享链接认证 (访客)                 │
└────────────────┬────────────────────────┘
                 ▼
┌─────────────────────────────────────────┐
│         加载应用与历史记录               │
│  ├─ MongoApp.findById()                 │
│  ├─ getChatItems() 加载历史             │
│  └─ 合并变量和上下文                    │
└────────────────┬────────────────────────┘
                 ▼
┌─────────────────────────────────────────┐
│         工作流执行引擎                   │
│  dispatchWorkFlow()                     │
│  ├─ 初始化运行时节点                    │
│  ├─ DAG 拓扑排序                        │
│  ├─ 循环执行节点                        │
│  │   ├─ AI 对话节点                     │
│  │   ├─ 知识库搜索节点                  │
│  │   ├─ 工具调用节点                    │
│  │   └─ ...                             │
│  └─ 流式 SSE 输出                       │
└────────────────┬────────────────────────┘
                 ▼
┌─────────────────────────────────────────┐
│           保存与统计                     │
│  ├─ saveChat() 保存对话                 │
│  ├─ 更新使用统计                        │
│  └─ 返回响应                            │
└─────────────────────────────────────────┘
```

### 5.3 知识库检索流程

```
用户查询
    │
    ▼
┌─────────────────────────────────────────┐
│         查询预处理                       │
│  ├─ 查询扩展 (queryExtension)           │
│  └─ 生成多个搜索词                       │
└────────────────┬────────────────────────┘
                 ▼
┌─────────────────────────────────────────┐
│         多路召回                         │
│  ├─ 向量搜索 (Embedding)                │
│  │   └─ getVectorsByText()              │
│  │   └─ vectorStore.search()            │
│  ├─ 全文搜索 (BM25)                     │
│  │   └─ MongoDB Text Index              │
│  └─ 结果合并                            │
└────────────────┬────────────────────────┘
                 ▼
┌─────────────────────────────────────────┐
│         重排序 (Rerank)                  │
│  ├─ 调用 Rerank 模型                    │
│  └─ 按相关性重新排序                    │
└────────────────┬────────────────────────┘
                 ▼
┌─────────────────────────────────────────┐
│         RRF 融合                         │
│  Score = Σ(weight × 1/(k + rank))       │
│  └─ 返回 Top-K 结果                     │
└─────────────────────────────────────────┘
```

## 7. 开发命令

```bash
# 开发
pnpm dev                    # 启动所有项目
cd projects/app && pnpm dev # 仅启动主应用

# 构建
pnpm build                  # 构建所有项目

# 测试
pnpm test                   # 运行测试
pnpm test:workflow          # 工作流测试

# 代码质量
pnpm lint                   # ESLint 检查
pnpm format-code            # Prettier 格式化

# 工具
pnpm create:i18n            # 生成国际化文件
pnpm gen:theme-typings      # 生成主题类型
```
