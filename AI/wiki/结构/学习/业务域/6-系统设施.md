# 业务域 6　系统设施

> 全局配置、授权、辅助能力。不属于具体业务但每个业务都用得到。

## 域内 domain 文件

| 文件 | 职责 | 范围 |
|---|---|---|
| `license.go` | License 授权 | 全局 |
| `setting.go` | 全局/KB 级设置 | 混合 |
| `icon.go` | 图标库（emoji + svg） | 全局 |
| `block_word.go` | 屏蔽词 | KB 级 |
| `announcement.go`（新增） | 门户更新公告 | 全局 |

## License 授权

`license` 表存企业版授权信息。当前二开已固定为企业版（见 `docs/二开日志/2025-10-25-License企业版固定实现.md`），所有高级功能开放。

## 设置的层级

```
全局 setting     —— 影响所有 kb
   ↓
KB-level setting —— 单个 kb 的覆盖
   ↓
App-level config —— 单个前台门户的样式/AI 配置
```

## 屏蔽词

- 黑名单：禁止输出包含这些词的内容
- 应用在 [[3-AI对话|Chat]] 输出层
