# 业务域 5　用户反馈

> 收集前台用户的"主动表达"和"被动行为"两类信号。

## 两类反馈

| 类别 | 主动？ | 表 | 域文件 |
|---|---|---|---|
| **评论** | ✅ 主动 | `user_comment` | `comment.go` |
| **文档点赞/反馈** | ✅ 主动 | `document_feedback` | `comment.go`（共用） |
| **AI 问答反馈** | ✅ 主动 | `conversation_message_feedback` | （在域 3） |
| **页面访问统计** | ❌ 被动 | `stat_pages` | `stat.go` |
| **小时聚合** | ❌ 被动 | `stat_page_hours` | `stat.go` |

## 评论功能

- 文档评论 → `user_comment`
- 评论审核（待审核 / 通过 / 拒绝）→ `user_comment.status`
- 用户显示格式："姓名(用户名)"（见 git log 2025-10-31）

## 统计功能

```
用户访问 App 页面
   ↓
share 路由打点 → stat_pages（实时表，每条访问一行）
   ↓
Consumer 定时任务聚合 → stat_page_hours（小时级聚合）
   ↓
Admin 看图表
```

## 维度

- 实时访客数
- 地域分布（IP → 地理位置）
- 热门文档
- 客户端类型
- 来源分析
