## 🏗️ 技术架构

### 技术栈

- **框架**: React 19 + TypeScript
- **构建工具**: Vite
- **UI 库**: Material-UI (MUI)
- **路由**: React Router v6
- **状态管理**: Redux Toolkit
- **HTTP 客户端**: Axios
- **代码规范**: ESLint + Prettier

### 目录结构

```
web/admin/
├── src/
│   ├── api/              # API 接口定义
│   ├── assets/           # 静态资源（图片、字体、样式）
│   ├── components/       # 公共组件
│   ├── constant/         # 常量定义
│   ├── hooks/            # 自定义 Hooks
│   ├── layouts/          # 布局组件
│   ├── pages/            # 页面组件
│   │   ├── login/        # 登录
│   │   ├── document/     # 文档管理
│   │   ├── conversation/ # 对话管理
│   │   ├── setting/      # 设置
│   │   ├── stat/         # 统计
│   │   ├── feedback/     # 反馈
│   │   ├── contribution/ # 贡献
│   │   └── release/      # 发布
│   ├── request/          # API 请求封装
│   ├── router.tsx        # 路由配置
│   ├── store/            # Redux 状态管理
│   ├── themes/           # 主题配置
│   └── utils/            # 工具函数
├── public/               # 公共资源
└── dist/                 # 构建输出
```

## 🚀 路由结构

### 主布局路由（带侧边栏和头部）

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | 文档管理 | 默认首页，文档列表 |
| `/setting` | 系统设置 | 系统配置页面 |
| `/contribution` | 贡献管理 | 文档贡献管理 |
| `/release` | 版本发布 | 版本管理 |
| `/stat` | 统计分析 | 数据统计和分析 |
| `/conversation` | 对话管理 | AI 对话记录 |
| `/feedback/:tab?` | 反馈管理 | 用户反馈（评价/评论） |

### 无侧边栏布局（需认证）

| 路径 | 页面 | 说明 |
|------|------|------|
| `/doc/editor/:id` | 文档编辑器 | 富文本编辑页面 |
| `/doc/editor/history/:id` | 历史版本 | 文档历史记录 |
| `/doc/editor/space` | 空间视图 | 文档空间管理 |

### 无侧边栏布局（无需认证）

| 路径 | 页面 | 说明 |
|------|------|------|
| `/login` | 登录页面 | 用户登录 |
| `/401` | 无权限页面 | 权限不足提示 |

## 📝 开发规范

### 组件命名
- 页面组件：PascalCase，如 `DocumentList`
- 普通组件：PascalCase，如 `CustomModal`
- 工具函数：camelCase，如 `formatDate`

### 文件组织
- 每个页面一个目录
- 复杂组件拆分子组件
- 相关逻辑抽取 hooks
- 常量和类型单独文件

### API 请求
- 统一使用 `request/` 目录下的封装
- 按模块分文件管理
- 类型定义完整
