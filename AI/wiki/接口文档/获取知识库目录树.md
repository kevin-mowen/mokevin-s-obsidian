# 获取知识库目录树

## 功能描述

获取指定知识库的文件夹目录树结构，返回嵌套的文件夹层级。用于上传文档前选择目标文件夹。

## API 接口

| 方法   | 路径                   |
| ---- | -------------------- |
| POST | `/api/v1/kb/folders` |

## 请求头

| Header | 必填 | 说明 |
|--------|------|------|
| Authorization | 是 | `Bearer <system_api_token>`，仅支持系统级 API Token（在系统配置 → 系统 API Token 中创建），KB 级别 Token 不可用。Token 需找管理员获取 |
| Content-Type | 是 | `application/json` |

## 请求参数

| 参数                | 类型     | 必填  | 说明                                                               | 示例                |
| ----------------- | ------ | --- | ---------------------------------------------------------------- | ----------------- |
| system_id         | string | 是   | 调用方系统编号                                                          | `"fastgpt"`       |
| agent_id          | string | 否   | 智能体 ID                                                           | `"bot_12345"`     |
| access_credential | string | 是   | 使用 notes_id + Secret 经 AES-256-GCM 加密后的 Base64 密文，Secret 需找管理员获取 | `"nGhJ8kL2mN..."` |
| kb_id             | string | 是   | 知识库 ID（从 `/api/v1/kb/list` 获取）                                   | `"3adce2bd-..."`  |

## 调用示例

```bash
curl -s -X POST http://localhost:18000/api/v1/kb/folders \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <system_api_token>" \
  -d '{
    "system_id": "fastgpt",
    "access_credential": "<AES-256-GCM加密后的Base64密文>",
    "kb_id": "be8666e3-7823-4e20-9f1c-f77ac604b47a"
  }' | jq
```

## 成功响应

```json
{
  "success": true,
  "code": 0,
  "message": "success",
  "data": [
    {
      "folder_id": "aaa-bbb-ccc",
      "folder_name": "文件夹1",
      "children": [
        {
          "folder_id": "ddd-eee-fff",
          "folder_name": "子文件夹",
          "children": []
        }
      ]
    },
    {
      "folder_id": "ggg-hhh-iii",
      "folder_name": "文件夹2",
      "children": []
    }
  ]
}
```

## 响应字段

| 字段 | 类型 | 说明 |
|------|------|------|
| data | array | 根级文件夹列表（嵌套结构） |
| data[].folder_id | string | 文件夹 ID，用于上传接口的 parent_id |
| data[].folder_name | string | 文件夹名称 |
| data[].children | array | 子文件夹列表（递归结构） |

## 权限过滤

返回的文件夹已经过 visitable 权限过滤，closed/partial（用户无权）的文件夹不会出现在目录树中。

## 错误码

| 错误码 | 说明 |
|--------|------|
| 401 | 非系统级 API Token 或 Token 无效 |
| 40001 | system_id、access_credential 或 kb_id 为空，或解密失败 |
| 40003 | 用户不存在或无权访问该知识库 |
