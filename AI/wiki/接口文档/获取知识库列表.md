# 获取用户有权限的知识库列表

## 功能描述

根据用户的 Notes ID（工号，通过 AES-256-GCM 加密传输），返回该用户有权限访问的所有知识库信息。

## API 接口

| 方法 | 路径 |
|------|------|
| POST | `/api/v1/kb/list` |

## 请求头

| Header        | 必填  | 说明                                                                                                   |
| ------------- | --- | ---------------------------------------------------------------------------------------------------- |
| Authorization | 是   | `Bearer <system_api_token>`，仅支持系统级 API Token（在系统配置 → 系统 API Token 中创建），KB 级别 Token 不可用。Token 需找管理员获取 |
| Content-Type  | 是   | `application/json`                                                                                   |

## 请求参数

| 参数 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| system_id | string | 是 | 调用方系统编号 | `"fastgpt"` |
| agent_id | string | 否 | 智能体平台的智能体/应用 ID，普通系统不传 | `"bot_12345"` |
| access_credential | string | 是 | 使用 notes_id + Secret 经 AES-256-GCM 加密后的 Base64 密文，Secret 需找管理员获取 | `"nGhJ8kL2mN..."` |

## 调用示例

```bash
curl -s -X POST http://localhost:18000/api/v1/kb/list \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <system_api_token>" \
  -d '{
    "system_id": "Z046X0",
    "access_credential": "<AES-256-GCM加密后的Base64密文>"
  }' | jq
```

## 成功响应

```json
{
  "success": true,
  "code": 0,
  "message": "success",
  "data": {
    "total": 2,
    "items": [
      {
        "kb_id": "3adce2bd-d5de-41a7-83aa-3b235be92b38",
        "kb_name": "产品知识库",
        "node_count": 42,
        "base_url": "http://wiki.example.com/node/xxx?kb=xxx"
      }
    ]
  }
}
```

## 响应字段

| 字段 | 类型 | 说明 |
|------|------|------|
| data.total | number | 知识库总数 |
| data.items[].kb_id | string | 知识库 UUID，用于其他接口的 kb_ids 传参 |
| data.items[].kb_name | string | 知识库名称 |
| data.items[].node_count | number | 文档数量 |
| data.items[].base_url | string | 知识库访问链接 |

## 认证与权限

1. **系统 Token 认证**：验证调用方身份，KB 级别 Token 返回 401
2. **用户身份解密**：使用系统 Token 的 Secret 解密 access_credential，得到明文 notes_id
3. **用户权限过滤**：根据解密后的 notes_id 用户的实际权限返回可访问的知识库列表

## 错误码

| 错误码 | 说明 |
|--------|------|
| 401 | 非系统级 API Token 或 Token 无效 |
| 40001 | system_id 或 access_credential 为空，或解密失败 |
| 40003 | 用户不存在（未通过 CAS 同步） |
