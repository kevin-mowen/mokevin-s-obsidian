# 文档上传

## 功能描述

通过 API 上传文件到指定知识库，自动完成文件解析、内容转换、创建文档节点。支持同名覆盖。如需发布，请使用 `/api/v1/kb/doc/publish` 接口。

## API 接口

| 方法   | 路径                  | Content-Type          |
| ---- | ------------------- | --------------------- |
| POST | `/api/v1/kb/upload` | `multipart/form-data` |

## 请求头

| Header | 必填 | 说明 |
|--------|------|------|
| Authorization | 是 | `Bearer <system_api_token>`，仅支持系统级 API Token（在系统配置 → 系统 API Token 中创建），KB 级别 Token 不可用。Token 需找管理员获取 |

## 请求参数（form-data）

| 参数           | 类型     | 必填  | 说明                                         | 示例               |
| ------------ | ------ | --- | ------------------------------------------ | ---------------- |
| system_id    | string | 是   | 调用方系统编号                                    | `"fastgpt"`      |
| agent_id     | string | 否   | 智能体 ID                                     | `"bot_12345"`    |
| access_credential | string | 是 | 使用 notes_id + Secret 经 AES-256-GCM 加密后的 Base64 密文，Secret 需找管理员获取 | `"nGhJ8kL2mN..."` |
| kb_id        | string | 是   | 目标知识库 ID                                   | `"be8666e3-..."` |
| parent_id    | string | 否   | 目标文件夹 ID，不传放根目录                            | `"019b48ef-..."` |
| doc_name     | string | 否   | 自定义文档标题，不传用文件名                             | `"数据库配置指南"`      |
| overwrite    | string | 否   | 同名文档是否覆盖，默认 `"false"`                      | `"true"`         |
| file         | File   | 是   | 上传的文件                                      | —                |

## 支持的文件类型

`.txt` `.md` `.xls` `.xlsx` `.doc` `.docx` `.pdf` `.html` `.pptx`

## 调用示例

```bash
curl -s -X POST http://localhost:18000/api/v1/kb/upload \
  -H "Authorization: Bearer <system_api_token>" \
  -F "system_id=fastgpt" \
  -F "access_credential=<AES-256-GCM加密后的Base64密文>" \
  -F "kb_id=be8666e3-782*******" \
  -F "parent_id=019b48ef-4ef******" \
  -F "doc_name=测试接口导入文件" \
  -F "file=@/Users/mokevin/Downloads/1.md" | jq
```

## 成功响应

```json
{
  "success": true,
  "code": 0,
  "message": "success",
  "data": {
    "doc_id": "019b6e86-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "doc_name": "数据库配置指南"
  }
}
```

## 响应字段

| 字段 | 类型 | 说明 |
|------|------|------|
| data.doc_id | string | 创建的文档 ID |
| data.doc_name | string | 文档标题 |

## 权限校验

1. **系统 Token 认证**：仅接受系统级 API Token
2. **用户身份解密**：使用系统 Token 的 Secret 解密 access_credential
3. **知识库访问权限**：校验解密后的 notes_id 用户是否有该 KB 的访问权限
3. **文件夹权限**：传了 parent_id 时，校验目标文件夹的 visitable 权限，用户无权的文件夹（partial/closed）不允许上传

## 错误码

| 错误码 | 说明 |
|--------|------|
| 401 | 非系统级 API Token 或 Token 无效 |
| 40001 | 必填参数为空、access_credential 解密失败或文件类型不支持 |
| 40003 | 用户不存在、无权访问该知识库或无权访问目标文件夹 |
| 50001 | 文件解析失败、内容转换失败或处理超时 |
