## 数据验证 SQL

### 验证数据一致性

```sql
-- panda-wiki 数据库
SELECT id, name, dataset_id FROM knowledge_bases;

-- raglite 数据库
SELECT id, name FROM datasets;
```

### 检查孤儿数据

```sql
-- 找出 raglite 中存在但 panda-wiki 中不存在的 dataset
-- 在 raglite 中执行
SELECT id FROM datasets
WHERE id NOT IN (
    SELECT dataset_id FROM panda_wiki.knowledge_bases
);
```
