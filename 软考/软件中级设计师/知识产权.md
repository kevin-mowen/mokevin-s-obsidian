# ⚖️ 知识产权与法律法规

## 一、著作权（Copyright）

### 权利类型

- 人身权：
	- 发表权：生前终身；生后，50年
	- 署名权、修改权、保护作品完整权：终身
- 财产权：受时间限时
	- 发行权
	- 展览权
- 《中华人民共和国著作权》规定，软件作品享有两类权利，一类是软件著作权的人身权（精神权利）；另一类是软件著作权的财产权（经济权利）

---

## 二、四大知识产权对比（高频考点）

| 类型      | 保护对象        | 保护期限                | 取得方式      | 关键特征      |
| ------- | ----------- | ------------------- | --------- | --------- |
| **著作权** | 文学、艺术、软件作品  | 人身权终身；财产权 50 年      | 自动产生（创作即得） | 表达形式      |
| **专利权** | 发明、实用新型、外观  | 发明 20 年；实用/外观 10 年  | 申请审查制     | 技术方案      |
| **商标权** | 商品/服务标志     | 10 年（可续展，每次 10 年）   | 注册取得      | 指示来源      |
| **商业秘密** | 经营/技术秘密     | 没有期限（保密期）           | 保密措施      | 不为公众所知    |

> 💡 **口诀**：著作 50 终身在，专利 20 实十外；商标 10 年可续展，秘密只要保得来。

## 三、职务作品 / 职务发明

| 类型 | 权利归属 | 备注 |
| ---- | -------- | ---- |
| 一般职务作品 | 作者，单位优先使用 2 年 | |
| 特殊职务作品（利用单位资源完成） | 单位享有，作者享署名权 | 软件作品多属此类 |
| 职务发明（利用单位物质条件） | 单位享有 | 发明人享署名权与奖励 |

## 四、侵权与合理使用

### 合理使用（不构成侵权）
- 个人学习、研究、欣赏
- 学校课堂教学使用
- 国家机关执行公务使用
- 介绍、评论某作品而适当引用
- 时事新闻报道

### 侵权判定要点
- 软件著作权侵权：**实质性相似 + 接触可能**
- 反向工程在我国一般 **不构成侵权**（但合同另有约定的除外）

## 五、标准化（常考冷点）

| 标准类型 | 代号 | 性质 |
| ---- | --- | ---- |
| 国际标准 | ISO、IEC | 国际通用 |
| 国家标准 | GB（强制）、GB/T（推荐） | 全国通用 |
| 行业标准 | YD、JB 等 | 行业内 |
| 地方标准 | DB | 地方内 |
| 企业标准 | Q | 企业内 |

> 💡 GB 与 GB/T 区分：**T = 推荐**，没 T 就是强制。
