### 自然连接

- 计算过程
	- 计算R✖️S；
	- 设R和S的公共属性是A<sub>1</sub> … A<sub>k</sub> ，挑选R✖️S中满足R.A<sub>1</sub> = S.A<sub>1</sub> … R.A<sub>k</sub> = S.A<sub>k</sub> 的那些元组；
	- 去掉R.A<sub>1</sub> = S.A<sub>1</sub> … R.A<sub>k</sub> = S.A<sub>k</sub> 这些列（保留R.A<sub>1</sub> … R.A<sub>k</sub> ）。
- 如果自然连接，没有一个完全相同的话，就是空集


### 左外连接、右外连接、全外连接

- 外连接运算是连接运算的扩展，可以处理由于连接运算而缺失的信息。

- 左外连接：取出左侧关系中所有与右侧关系中任一元组都不匹配的元组，用空值null 充填所有来自右侧关系的属性，构建新的元组
- 右外连接：与左外连接相反
- 全外连接：完成左外连接和右外连接的操作

## **基础概念（做题必备）**

- **函数依赖**（FD） X → Y：属性集 X 的值**唯一决定** Y。
    
- **完全函数依赖**：在 X → Y 中，Y 依赖 X 的**所有**属性；去掉 X 的任何一部分都不能决定 Y。
    
- **部分函数依赖**：在 X → Y 中，Y 只依赖 X 的**一部分**（常对应复合主键中的一部分）。
    
- **传递函数依赖**：X → Z 且 Z → Y，则 X → Y 为传递依赖（Z 不含在 X 中）。
    
- **候选键**：最小超键。
    
- **主属性**：出现在某个候选键中的属性。
    
- **非主属性**：不在任何候选键中的属性。
    
- **关键判断口诀**：
    
    - 2NF：**非主属性**不得**部分依赖**候选键。
        
    - 3NF：**非主属性**不得**传递依赖**候选键。
        
    - BCNF：**任一**非平凡依赖 X → Y 中，X 必须是**超键**。
        
    - 4NF：消除**非平凡且非函数**的**多值依赖**。
        
    - 5NF：处理**连接依赖**（分解再自然连接能还原）。

### BC范式（BCNF）

- 设R是一个关系模式，F是它的依赖集，R属于BCNF，当且仅当其F中每个依赖的决定因素必定包含R的某个后选码
- 所有非主属性对每一个码都是完全函数依赖
- 所有的主属性对每一个不包含它的码，也是完全函数依赖
- 没有任何属性完全依赖于非码的任何一组属性
