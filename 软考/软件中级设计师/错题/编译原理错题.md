# ⚙️ 编译原理错题

> 关联：[[程序设计语言基础与编译原理知识总结]]

## 编译各阶段处理的错误类型（高频考点）

| 阶段 | 处理的错误 |
| ---- | --------- |
| **词法分析** | 非法字符、单词拼写错误等 |
| **语法分析** | 标点符号错误、表达式中缺少操作数、括号不匹配等 |
| **静态语义分析** | 运算符与运算对象类型不合法等错误 |
| **目标代码生成 / 执行阶段** | **动态语义错误**（如除零、数组越界） |

> 💡 **记忆**：
> - 拼写 → 词法
> - 括号、标点 → 语法
> - 类型不匹配 → 语义（静态）
> - 除零、越界 → 运行时（动态）
