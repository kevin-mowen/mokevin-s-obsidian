# 🗄️ 数据库错题

> 关联：[[数据库]]

## 分布式数据库的四大性质

| 性质 | 含义 |
| ---- | ---- |
| **共享性** | 数据存储在不同结点，可被共享访问 |
| **自治性** | 每个结点对本地数据**能独立管理** |
| **可用性** | 某场地故障时，可使用其他场地的副本，**整个系统不瘫痪** |
| **分布性** | 数据在不同场地上分布存储 |

> 💡 **记忆口诀**：分布共享、各管自家、坏一不亡。
