# 💻 计算机组成与体系结构错题

> 关联：[[💻 计算机系统硬件与组成原理]]

## 计算机性能指标

| 指标 | 含义 |
| ---- | ---- |
| **吞吐量** | 给定时间内系统所能处理的任务数量 |
| **响应时间** | 系统对请求作出响应的时间 |
| **容量** | 存储器所能存储的全部信息量 |
| **负载** | 系统能够承受的最大任务数 |

## 输入输出（I/O）控制方式

| 类型 | CPU 与 I/O 关系 | 数据传输 | 单位 |
| ---- | -------------- | ------- | ---- |
| **程序查询方式** | 串行；CPU 全程参与 | CPU 把数据放入内存 | 字 |
| **中断驱动方式** | 可并行；I/O 主动中断 CPU | CPU 把数据放入内存 | 字 |
| **DMA 直接存储器存取** | 可并行；CPU 仅在开始/结束干预 | 外设直接放入内存 | 块 |

## I/O 接口同步与异步

- CPU ↔ 内存储器：**同步**（严格时序）
- CPU ↔ PCI 总线设备：**同步**（PCI 协议定义严格时序）
- CPU ↔ I/O 接口：通常**同步**（按系统时钟周期）
- I/O 接口 ↔ 打印机：**基于缓存池的异步**

## 存储器类型

| 类型 | 全称 | 特点 |
| ---- | ----- | ---- |
| **DRAM** | 动态随机存取 | 集成率高、功耗高、需周期刷新；与 CPU 直接交换；**断电丢失** |
| **SRAM** | 静态随机存取 | 集成率低、功耗低、不需刷新；通电时数据恒久保持；做 Cache |
| **FLASH** | 闪存 | 介于 EPROM 和 EEPROM 之间；可电擦除整块 |
| **EEPROM** | 电擦除可编程 ROM | 可读可改写；按字节擦除 |
