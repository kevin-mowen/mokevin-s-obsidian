# 🌐 计算机网络错题

## 网关协议

- **内部网关协议（IGP）**：RIP、OSPF、IS-IS、IGRP、EIGRP
- **外部网关协议（EGP）**：BGP

## 网络与远程登录协议

| 协议 | 所属层 | 用途 |
| ---- | ----- | ---- |
| **IGMP** | 网络层 | 组播协议；**不能**实现应用层远程登录 |
| **SSH** | 应用层 | 较可靠的安全协议，专门用于远程登录会话和其他网络服务 |
| **Telnet** | 应用层 | TCP/IP 中标准的远程登录服务（不安全） |
| **RFB** | 应用层 | 用于远程访问图形用户界面（VNC 用） |

## Windows / Linux 命令对照

| 命令 | 用途 | 备注 |
| ----- | ---- | ---- |
| `ipconfig` | 查看本机 IP 信息 | Windows |
| `tracert` | 路由跟踪 | Windows，对应 Linux 的 `traceroute` |
| `netstat` | 监控 TCP/IP 网络连接 | 显示路由表、连接、接口状态 |
| `nslookup` | 域名查询 | 查询 DNS 解析记录 |
