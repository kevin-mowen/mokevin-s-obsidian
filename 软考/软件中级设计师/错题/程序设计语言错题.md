# 🐍 程序设计语言错题

## Python 数据类型

### 不可变数据（Immutable）
- **Number**（数字）
- **String**（字符串）
- **Tuple**（元组）

### 可变数据（Mutable）
- **List**（列表）
- **Dictionary**（字典）
- **Set**（集合）

### 容器对比

| 类型 | 标识 | 特点 |
| ---- | ---- | ---- |
| `tuple` | `()` | 类似 list，但**不可变**；可作字典的 key |
| `list` | `[]` | 有序对象集合；可包含字符、数字、字符串、嵌套列表 |
| `dict` | `{}` | 由索引 key 和对应 value 组成；**3.7+ 保持插入顺序** |
| `set` | `{}` | 无序、不重复元素集合 |

> 💡 易混淆：
> - 字典与集合都用 `{}`，但**空 `{}` 默认是字典**，空集合需 `set()`。
> - tuple 不可变指其引用不可变，**内部可变对象（如 list）仍可被修改**。
