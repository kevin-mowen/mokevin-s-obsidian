# 🔒 信息安全与加密错题

## 信息安全的五个基本要素

| 要素       | 含义                         |
| -------- | -------------------------- |
| **机密性**  | 信息不暴露给未授权的实体或进程            |
| **完整性**  | 只有授权者能修改数据，并能判别数据是否被篡改     |
| **可用性**  | 授权实体在需要时可访问数据；攻击者不能阻止授权者工作 |
| **可控性**  | 可以控制授权范围内的信息流向及行为方式        |
| **可审查性** | 出现安全问题时可提供调查依据和手段          |

## 加密算法分类

| 类型               | 特点                | 常见算法                 |
| ---------------- | ----------------- | -------------------- |
| **公钥加密 / 非对称加密** | 公钥加密、私钥解密；密钥成对    | RSA、ElGamal、背包算法、ECC |
| **流加密**（对称）      | 加解密用相同伪随机密钥流，逐位运算 | RC4、ORYX、SEAL        |
| **分组加密**（对称）     | 明文分块，使用相同密钥加解密    | DES、3DES、AES         |
| **消息摘要 / 数字摘要**  | 单向 Hash，固定长度      | MD5、SHA 系列、MAC       |

> 💡 **口诀**：对称速度快但分发难；非对称分发好但速度慢；HTTPS = 非对称传密钥 + 对称传数据。
