# 📐 系统配置与性能评价

> 状态：🟡 骨架（待补充）。

## 一、系统结构分类（Flynn 分类）

| 类型 | 说明 | 例子 |
| ---- | ---- | ---- |
| **SISD** | 单指令单数据 | 传统冯·诺依曼计算机 |
| **SIMD** | 单指令多数据 | 阵列处理机、GPU |
| **MISD** | 多指令单数据 | 罕见 |
| **MIMD** | 多指令多数据 | 多核 / 多处理器系统 |

## 二、计算机性能指标

| 指标 | 含义 |
| ---- | ---- |
| 主频 | CPU 时钟频率 |
| CPI | 每指令周期数 |
| MIPS | 每秒百万条指令 |
| MFLOPS | 每秒百万次浮点运算 |
| 吞吐量 | 单位时间处理任务数 |
| 响应时间 | 请求 → 响应的时间 |
| 利用率 | 资源被使用比例 |

## 三、性能评价方法

- **时钟频率法**：主频对比
- **指令执行速度法**：MIPS、MFLOPS
- **基准程序法（Benchmark）**：
  - 整数：Dhrystone
  - 浮点：Whetstone、Linpack
  - 综合：SPEC、TPC

## 四、阿姆达尔定律（Amdahl's Law）

$$S = \frac{1}{(1-P) + \frac{P}{N}}$$

- P：可并行部分占比
- N：处理器数量
- S：加速比

> 💡 即使 N → ∞，加速比上限为 $\frac{1}{1-P}$。

## 五、系统可靠性

| 系统类型 | 公式 |
| ---- | ---- |
| 串联 | $R = R_1 \times R_2 \times \cdots \times R_n$ |
| 并联 | $R = 1 - \prod (1 - R_i)$ |
| 混合 | 先简化子系统再组合 |
