## 🟢 1. 直接插入排序（Insertion Sort）

**思路**：依次将元素插入到前面已排序序列中。

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr

print(insertion_sort([5, 2, 9, 1, 5, 6]))
```

**第一轮（i=1）**：比较 `5` 和 `2`，交换 → `[2, 5, 9, 1, 5, 6]`
**最终输出**：`[1, 2, 5, 5, 6, 9]`

## 🟡 2. 希尔排序（Shell Sort）

**思路**：按间隔 `gap` 进行分组插入排序，逐渐缩小 `gap`。

```python
def shell_sort(arr):
    gap = len(arr) // 2
    while gap > 0:
        for i in range(gap, len(arr)):
            temp = arr[i]
            j = i
            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap
            arr[j] = temp
        gap //= 2
    return arr

print(shell_sort([5, 2, 9, 1, 5, 6]))
```

**第一轮（gap=3）**：比较并交换 `(5, 1)` → `[1, 2, 9, 5, 5, 6]`
**最终输出**：`[1, 2, 5, 5, 6, 9]`

## 🟣 3. 选择排序（Selection Sort）

**思路**：每一轮选择最小值放到当前未排序部分的最前面。

```python
def selection_sort(arr):
    for i in range(len(arr)):
        min_idx = i
        for j in range(i + 1, len(arr)):
            if arr[j] < arr[min_idx]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr

print(selection_sort([5, 2, 9, 1, 5, 6]))
```

**第一轮**：遍历，最小是 `1`，与第一个元素交换 → `[1, 2, 9, 5, 5, 6]`
**最终输出**：`[1, 2, 5, 5, 6, 9]`

## 🔵 4. 堆排序（Heap Sort）

**思路**：构造最大堆 → 取堆顶 → 调整堆。

```python
def heapify(arr, n, i):
    largest = i
    l, r = 2*i + 1, 2*i + 2
    if l < n and arr[l] > arr[largest]:
        largest = l
    if r < n and arr[r] > arr[largest]:
        largest = r
    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)

def heap_sort(arr):
    n = len(arr)
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    for i in range(n - 1, 0, -1):
        arr[0], arr[i] = arr[i], arr[0]
        heapify(arr, i, 0)
    return arr

print(heap_sort([5, 2, 9, 1, 5, 6]))
```

**第一轮**：建堆后 `[9,5,6,1,2,5]` → 堆顶与末尾交换 → `[5,5,6,1,2,9]`
**最终输出**：`[1, 2, 5, 5, 6, 9]`

## 🔴 5. 冒泡排序（Bubble Sort）

**思路**：相邻两两比较，把大的往后"冒"。

```python
def bubble_sort(arr):
    for i in range(len(arr) - 1):
        for j in range(len(arr) - 1 - i):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr

print(bubble_sort([5, 2, 9, 1, 5, 6]))
```

**第一轮**：
- (5,2) → `[2,5,9,1,5,6]`
- (9,1) → `[2,5,1,9,5,6]`
- (9,5) → `[2,5,1,5,9,6]`
- (9,6) → `[2,5,1,5,6,9]`
最大值 `9` 已沉底。
**最终输出**：`[1, 2, 5, 5, 6, 9]`

## 🟠 6. 快速排序（Quick Sort）

**思路**：选 pivot，把数组分成左中右三部分递归排序。

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    mid  = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + mid + quick_sort(right)

print(quick_sort([5, 2, 9, 1, 5, 6]))
```

**第一轮**：选中枢 `pivot = 9`，左 `[5,2,1,5,6]`，中 `[9]`，右 `[]` → `[5,2,1,5,6,9]`
**最终输出**：`[1, 2, 5, 5, 6, 9]`

## 🟢 7. 归并排序（Merge Sort）

**思路**：分治 → 拆分成单个 → 逐步合并。

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i]); i += 1
        else:
            result.append(right[j]); j += 1
    result += left[i:]
    result += right[j:]
    return result

print(merge_sort([5, 2, 9, 1, 5, 6]))
```

**第一轮（拆为左右半边）**：
- 左 `[5,2,9]` → `[2,5,9]`
- 右 `[1,5,6]` → `[1,5,6]`
- 合并后 `[1,2,5,5,6,9]`
**最终输出**：`[1, 2, 5, 5, 6, 9]`
