## 🗓️ 五、学习计划建议

| 阶段 | 时长 | 内容 |
| ---- | ---- | ---- |
| **基础** | 4 周 | 计算机基础（硬件/操作系统/网络）+ 数据结构 |
| **进阶** | 3 周 | 数据库 + 软件工程 + 面向对象 + 设计模式 |
| **专项** | 2 周 | 信息安全 + 知识产权 + 性能评价 |
| **真题** | 3 周 | 上午真题 + 下午案例分析 |
| **冲刺** | 1 周 | 错题回顾 + 易混知识点对比 |
