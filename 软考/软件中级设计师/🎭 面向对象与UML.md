# 🎭 面向对象与 UML

> 状态：🟡 骨架（待补充）。错题积累见 [[错题/面向对象与UML错题]]。

## 一、面向对象基础

### 1. 三大特征
- **封装（Encapsulation）**
- **继承（Inheritance）**
- **多态（Polymorphism）**

### 2. 多态分类（4 种）

| 类型 | 说明 |
| ---- | ---- |
| 参数多态 | 泛型；最纯粹 |
| 包含多态 | 子类型化；运行时检查 |
| 过载多态 | 重载 |
| 强制多态 | 类型自动转换 |

### 3. 类间关系
- 关联（Association）、聚合（Aggregation）、组合（Composition）
- 依赖（Dependency）、泛化（Generalization）、实现（Realization）

## 二、UML 图分类

### 1. 结构图（5 种）
- **类图**、对象图、组件图、部署图、包图

### 2. 行为图（4 种）
- **用例图**、活动图、状态图、（交互图）

### 3. 交互图（2 种）
- **序列图（顺序图）**、协作图

> 💡 类图是 UML 的核心；序列图与协作图描述同一信息（时间 vs 关系）。

## 三、UML 类图符号

| 符号 | 含义 |
| ---- | ---- |
| `+` | public |
| `-` | private |
| `#` | protected |
| `~` | package |
| `斜体` | 抽象类 / 抽象方法 |
| `下划线` | 静态成员 |

### 关系箭头

| 关系 | 符号 |
| ---- | ---- |
| 关联 | 实线 |
| 聚合 | 实线 + 空心菱形 |
| 组合 | 实线 + 实心菱形 |
| 依赖 | 虚线 + 箭头 |
| 泛化（继承） | 实线 + 空心三角箭头 |
| 实现 | 虚线 + 空心三角箭头 |

## 四、面向对象开发流程

| 阶段 | 主要活动 |
| ---- | -------- |
| OOA | 认定对象、组织对象、对象间相互作用、对象的操作 |
| OOD | 识别类与对象、定义属性、定义服务、识别关系、识别包 |
| OOP | 选择 OOPL，编码实现 |
| OOT | 算法层、类层、模板层、系统层 |

## 五、设计原则（SOLID + GRASP）

### SOLID
- **S**RP：单一职责
- **O**CP：开闭原则
- **L**SP：里氏替换
- **I**SP：接口隔离
- **D**IP：依赖倒置

详见 [[设计模式]]。
