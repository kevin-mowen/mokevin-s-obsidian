# 🖥️ 操作系统

> 状态：🟡 骨架（待补充）。错题积累见 [[错题/操作系统错题]]。

## 一、操作系统概述
- 五大功能：进程管理、存储管理、设备管理、文件管理、作业管理
- 类型：批处理、分时、实时、网络、分布式、嵌入式

## 二、进程管理
### 1. 进程状态
- 三态模型：就绪 → 运行 → 阻塞
- 五态模型：增加新建、终止
### 2. 进程同步与互斥
- PV 操作 / 信号量
- 经典问题：生产者-消费者、读者-写者、哲学家进餐
### 3. 进程调度算法
- FCFS、SJF、优先级、时间片轮转、多级反馈队列
### 4. 死锁
- 必要条件：互斥、占有并等待、不可剥夺、循环等待
- 处理：预防、避免（银行家算法）、检测、解除

## 三、存储管理
### 1. 分区存储
- 固定分区、可变分区、首次适应、最佳适应、最坏适应
### 2. 页式 / 段式 / 段页式
- 逻辑地址 → 物理地址转换
- 页表、TLB（快表）
### 3. 虚拟存储
- 局部性原理
- 页面置换算法：OPT、FIFO、LRU、Clock
- 缺页率、抖动

## 四、设备管理
- I/O 控制方式（详见 [[错题/计算机组成与体系结构错题#输入输出（I/O）控制方式]]）
- SPOOLing 假脱机
- 磁盘调度：FCFS、SSTF、SCAN、C-SCAN

## 五、文件管理
- 文件结构、目录结构
- 文件分配：连续、链接、索引
- 空闲空间管理：位图、空闲链表

## 六、作业管理
- 作业调度算法
- 用户接口
