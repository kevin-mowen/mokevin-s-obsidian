# 🔍 IP 地址知识点详细补强

> 补充资料 | 用于加强 IP 地址和子网掩码部分 | 真题高频考点

---

## 一、IPv4 地址基础

### 1️⃣ IP 地址的组成

**定义**：IPv4 地址是 32 位的二进制数，通常用四个十进制数（0-255）表示

```
IP 地址 = 32 位二进制数
表示方法：a.b.c.d (a, b, c, d ∈ [0, 255])

例如：192.168.1.100
转换为二进制：11000000.10101000.00000001.01100100
```

### 2️⃣ IP 地址的五种分类

#### A 类地址
- **首位**：0（二进制）
- **范围**：1.0.0.0 ~ 126.255.255.255
- **子网掩码**：255.0.0.0（/8）
- **网络位**：8 位，**主机位**：24 位
- **可用网络数**：126（0 和 127 保留）
- **每个网络的主机数**：2^24 - 2 = 16,777,214 个
- **用途**：大型网络（如国家级、全球组织）
- **记忆**：第一段 1-126

#### B 类地址
- **首两位**：10（二进制）
- **范围**：128.0.0.0 ~ 191.255.255.255
- **子网掩码**：255.255.0.0（/16）
- **网络位**：16 位，**主机位**：16 位
- **可用网络数**：2^14 = 16,384 个
- **每个网络的主机数**：2^16 - 2 = 65,534 个
- **用途**：中型网络（如大企业、大学）
- **记忆**：第一段 128-191

#### C 类地址
- **首三位**：110（二进制）
- **范围**：192.0.0.0 ~ 223.255.255.255
- **子网掩码**：255.255.255.0（/24）
- **网络位**：24 位，**主机位**：8 位
- **可用网络数**：2^21 = 2,097,152 个
- **每个网络的主机数**：2^8 - 2 = 254 个
- **用途**：小型网络（如家庭、小企业）
- **记忆**：第一段 192-223

#### D 类地址（多播/组播）
- **首四位**：1110（二进制）
- **范围**：224.0.0.0 ~ 239.255.255.255
- **用途**：多播地址（一对多通信）
- **例子**：224.0.0.5（OSPF 路由协议多播地址）

#### E 类地址（保留）
- **首四位**：1111（二进制）
- **范围**：240.0.0.0 ~ 255.255.255.255
- **用途**：保留用途，目前未分配

## 二、私有 IP 地址（重要！）

在局域网（LAN）中使用，不在互联网上路由：

| 地址类别 | 私有地址范围 | 子网掩码 | 用途 |
|---------|----------|--------|------|
| A 类私有 | 10.0.0.0 ~ 10.255.255.255 | 255.0.0.0 | 大型私有网络 |
| B 类私有 | 172.16.0.0 ~ 172.31.255.255 | 255.255.0.0 | 中型私有网络 |
| C 类私有 | 192.168.0.0 ~ 192.168.255.255 | 255.255.255.0 | 小型私有网络（家庭/小企业） |

**特殊地址**：
- 127.0.0.1：本地回环地址（localhost）
- 0.0.0.0：默认路由
- 255.255.255.255：广播地址（整个网络）

## 三、子网掩码详解

### 什么是子网掩码？

**定义**：用来区分 IP 地址中哪些位是网络地址，哪些位是主机地址的 32 位掩码

**特性**：
- 由连续的 1 和连续的 0 组成
- 1 对应网络部分，0 对应主机部分
- 用 AND 运算与 IP 地址结合

### 常见子网掩码速查表

| 掩码位数 | 十进制表示 | 二进制（最后 8 位） | 可容纳主机数 |
|---------|-----------|-----------------|----------|
| /8 | 255.0.0.0 | 00000000 | 16,777,214 |
| /16 | 255.255.0.0 | 00000000 | 65,534 |
| /24 | 255.255.255.0 | 00000000 | 254 |
| /25 | 255.255.255.128 | 10000000 | 126 |
| /26 | 255.255.255.192 | 11000000 | 62 |
| /27 | 255.255.255.224 | 11100000 | 30 |
| /28 | 255.255.255.240 | 11110000 | 14 |
| /29 | 255.255.255.248 | 11111000 | 6 |
| /30 | 255.255.255.252 | 11111100 | 2（常用于点对点链路） |
| /31 | 255.255.255.254 | 11111110 | 0（特殊用途） |
| /32 | 255.255.255.255 | 11111111 | 1（单个主机） |

## 四、IP 地址的三个重要地址

### 1️⃣ 网络地址（Network Address）

**定义**：整个子网的标识，主机部分全为 0

**计算方法**：
```
网络地址 = IP地址 AND 子网掩码
```

**例子**：
```
IP: 192.168.1.130
掩码: 255.255.255.128 (/25)

IP 二进制：     192.168.1.10000010
掩码二进制：    255.255.255.10000000
AND 运算：      192.168.1.10000000
网络地址：      192.168.1.128
```

### 2️⃣ 广播地址（Broadcast Address）

**定义**：整个子网中用于广播的地址，主机部分全为 1

**计算方法**：
```
广播地址 = 网络地址 + (2^主机位数 - 1)
或
广播地址 = IP地址 OR (NOT 子网掩码)
```

**例子**：
```
网络地址：192.168.1.128
主机位数：7（/25 表示网络位 25，主机位 32-25=7）
广播地址：192.168.1.128 + (2^7 - 1) = 192.168.1.255

OR 方法：
网络地址：192.168.1.10000000
掩码取反：255.255.255.01111111
OR 运算：192.168.1.11111111
广播地址：192.168.1.255
```

### 3️⃣ 可用主机地址

**定义**：能分配给主机的地址范围

**计算方法**：
```
可用主机数 = 2^主机位数 - 2
可用范围：从 (网络地址 + 1) 到 (广播地址 - 1)
```

**例子**：
```
网络地址：192.168.1.128
广播地址：192.168.1.255
主机位数：7

可用主机数：2^7 - 2 = 128 - 2 = 126
可用范围：192.168.1.129 ~ 192.168.1.254
```

## 五、重要的计算公式

### 公式 1：根据子网掩码判断网络位数

```
掩码位数 = 掩码中 1 的个数

例如：255.255.255.128
二进制：11111111.11111111.11111111.10000000
1 的个数：25 个
所以是 /25（25位网络，7位主机）
```

### 公式 2：主机位数计算

```
主机位数 = 32 - 网络位数
或
主机位数 = 掩码中 0 的个数
```

### 公式 3：计算子网数

```
子网数 = 2^(新增子网位数)

例如：将 C 类地址（原 /24）划分为 /26 的子网
新增子网位数 = 26 - 24 = 2
子网数 = 2^2 = 4 个子网
```

### 公式 4：计算每个子网的主机数

```
每个子网的主机数 = 2^主机位数 - 2
```

## 六、IP 地址计算的步骤方法

### 例题 1：标准计算

**题目**：给定 IP = 172.16.5.130，掩码 = 255.255.255.192（/26），求网络地址、广播地址、可用主机数、可用范围

**解答步骤**：

```
第 1 步：确定网络位和主机位
掩码 /26 = 11111111.11111111.11111111.11000000
网络位：26 位，主机位：6 位

第 2 步：计算网络地址
IP 二进制（最后 8 位）：10000010
掩码二进制（最后 8 位）：11000000
AND 运算：10000000 = 128

网络地址：172.16.5.128

第 3 步：计算广播地址
主机位数：6
广播地址：172.16.5.128 + (2^6 - 1) = 172.16.5.128 + 63 = 172.16.5.191

第 4 步：计算可用主机数
可用主机数：2^6 - 2 = 64 - 2 = 62 个

第 5 步：计算可用范围
从 172.16.5.129 到 172.16.5.190
共 62 个地址
```

### 例题 2：子网划分

**题目**：将网络 192.168.1.0/24 划分为 4 个等大小的子网，每个子网的掩码是多少？

**解答步骤**：

```
第 1 步：确定需要增加的子网位数
划分成 4 个子网 = 2^2
需要增加 2 位子网位数

第 2 步：计算新的掩码
原掩码：/24
新掩码：/24 + 2 = /26

掩码二进制：11111111.11111111.11111111.11000000
掩码十进制：255.255.255.192

第 3 步：列出 4 个子网
子网 1：192.168.1.0/26（192.168.1.0 ~ 192.168.1.63）
子网 2：192.168.1.64/26（192.168.1.64 ~ 192.168.1.127）
子网 3：192.168.1.128/26（192.168.1.128 ~ 192.168.1.191）
子网 4：192.168.1.192/26（192.168.1.192 ~ 192.168.1.255）

每个子网的可用主机数：2^6 - 2 = 62 个
```

## 七、易混点与记忆技巧

### 🔴 常见错误

| 错误 | 正确做法 | 原因 |
|------|--------|------|
| 网络地址分配给主机 | 网络地址不能用 | 网络地址代表整个子网 |
| 广播地址分配给主机 | 广播地址不能用 | 广播地址用于子网内广播 |
| 遗漏 -2 | 一定要减去 2 | 除了网络地址和广播地址 |
| 掩码中出现不连续的 1 和 0 | 掩码必须是连续的 1 + 连续的 0 | 这是掩码的定义 |
| 混淆 /25 和 255.255.255.128 | 它们是等价的 | /25 表示 25 位网络 |

### 💡 记忆技巧

| 技巧 | 说明 |
|------|------|
| **A/B/C 首位记忆** | A(0)、B(10)、C(110) 看二进制 |
| **A/B/C 范围记忆** | A(1-126)、B(128-191)、C(192-223) |
| **私有地址记忆** | 10.x.x.x、172.16-31.x.x、192.168.x.x |
| **掩码与点数关系** | /8→255.0.0.0、/16→255.255.0.0、/24→255.255.255.0 |
| **2 的幂次** | 2^7=128、2^8=256、2^6=64、2^5=32、2^4=16 |
| **主机数公式** | 2^主机位数 - 2（减去网络地址和广播地址） |

## 八、高频考题模板

### 模板 1：给定 IP 和掩码，求网络地址、广播地址

```
题目类型：
Q: IP=X.X.X.X，掩码=255.255.255.Y，求网络地址、广播地址、可用主机数

答题步骤：
1. 确定掩码位数（从 Y 推断，或直接给出 /n）
2. IP AND 掩码 → 网络地址
3. 网络地址 + (2^主机位 - 1) → 广播地址
4. 2^主机位 - 2 → 可用主机数
```

### 模板 2：子网划分

```
题目类型：
Q: 将网络 X.X.X.0/24 划分为 N 个相等的子网

答题步骤：
1. 计算需要的子网位数：log2(N)
2. 新掩码 = 原掩码 + 子网位数
3. 列出每个子网的范围
```

### 模板 3：判断两个 IP 是否在同一网络

```
题目类型：
Q: IP1=X.X.X.X，IP2=Y.Y.Y.Y，掩码=Z，是否在同一网络？

答题步骤：
1. IP1 AND 掩码 = 网络地址1
2. IP2 AND 掩码 = 网络地址2
3. 比较：网络地址1 == 网络地址2？
```

## 九、快速计算技巧（应急用）

### 掩码最后一位数的秘诀

```
掩码最后 8 位的 AND 值：
/24 = 0（全 0）= 256 可能性
/25 = 128
/26 = 192（128+64）
/27 = 224（128+64+32）
/28 = 240（128+64+32+16）
/29 = 248（128+64+32+16+8）
/30 = 252（128+64+32+16+8+4）

记忆：从 128 开始，逐个加上：64、32、16、8、4、2、1
```

### 快速判断 IP 分类

```
看第一个数字（十进制）：
1-126：A 类
128-191：B 类
192-223：C 类
224-239：D 类（多播）
240-255：E 类（保留）
```

## 十、实战练习题

### 练习 1

**题目**：IP = 10.20.30.40，掩码 = 255.255.255.0，这是几类地址？是私有还是公网？网络地址是什么？

**答案**：
- A 类（首位 10）
- 私有地址（10.x.x.x）
- 网络地址：10.20.30.0

### 练习 2

**题目**：IP = 172.16.100.200，掩码 = 255.255.0.0，广播地址是什么？

**答案**：
- 网络地址：172.16.0.0
- 广播地址：172.16.255.255

### 练习 3

**题目**：IP = 192.168.1.100，掩码 = 255.255.255.240，可用主机数是多少？

**答案**：
- 掩码 /28
- 主机位：32 - 28 = 4
- 可用主机数：2^4 - 2 = 14 个
