# 🎭 UML 速查卡

---

## 第一层：面向对象基础

| 特性 | 含义 | 例子 |
|------|------|------|
| 封装 | 隐藏内部实现，提供公共接口 | private、protected、public |
| 继承 | 子类继承父类的属性和方法 | 泛化关系 |
| 多态 | 同一接口，不同实现 | 方法重载、重写 |

---

## 第二层：UML 关系（重点！）

### 六大关系强度对比

| 关系 | 符号 | 含义 | 强度 |
|------|------|------|------|
| 泛化 | 空心三角形 | 继承关系 | **最强** |
| 实现 | 空心三角形+虚线 | 实现接口 | **强** |
| 组合 | 实心菱形 | 整体包含部分，部分不能独立 | **强** |
| 聚合 | 空心菱形 | 整体包含部分，部分可独立 | **中** |
| 关联 | 箭头 | 一个类使用另一个类 | **弱** |
| 依赖 | 虚线箭头 | 临时性使用 | **最弱** |

**记忆技巧**：菱形强，箭头弱；实心强，空心弱

## 第三层：UML 图形详解

### 类图元素

```
+------------------+
|    ClassName     |  ← 类名
+------------------+
| - attr1: int     |  ← 属性 (- 私有，# 保护，+ 公有)
| # attr2: String  |
| + attr3: boolean |
+------------------+
| + method1()      |  ← 方法
| - method2()      |
+------------------+
```

### 时序图基本元素

- **对象**：在顶部显示 `对象名:类名`
- **消息**：对象之间的箭头
  - 同步消息：实心箭头 →
  - 异步消息：半箭头 ⤓
  - 返回消息：虚线箭头 ← ← ←
- **激活框**：对象处于活跃状态的时间段

### 状态图

- **状态**：圆角矩形
- **事件/转移**：带标签的箭头
- **初始状态**：实心圆
- **终止状态**：空心圆内实心圆

---

## 易混点

❌ **不要混淆**：
- 组合 vs 聚合：关键是"部分能否独立"
- 泛化 vs 实现：泛化是 is-a，实现是 can-do
- 关联 vs 依赖：关联是长期，依赖是临时
