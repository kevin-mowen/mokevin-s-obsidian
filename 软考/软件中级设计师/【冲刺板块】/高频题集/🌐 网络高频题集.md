## 高频考点 1：OSI 七层模型与 TCP/IP

**真题出现频率**：每年 2-3 题

### 典型题 1.1
**题目**：OSI 七层模型中，DNS 工作在哪一层？
- A) 物理层
- B) 网络层
- C) 传输层
- D) 应用层

**答案**：D  
**解析**：DNS（Domain Name System）是域名解析服务，提供应用级服务，工作在应用层（第 7 层）。虽然 DNS 使用 UDP 协议（传输层），但 DNS 本身是应用层协议。  
**记忆要点**：应用层协议包括：HTTP、HTTPS、DNS、FTP、SMTP、Telnet 等  
**相似题**：检查各种应用层协议的定义

## 高频考点 2：IP 地址与子网

**真题出现频率**：每年 1-2 题

### 典型题 2.1
**题目**：给定 IP 地址 192.168.1.130，子网掩码 255.255.255.128，求该 IP 所在的网络地址。
- A) 192.168.1.0
- B) 192.168.1.128
- C) 192.168.1.255
- D) 192.168.2.0

**答案**：B  
**解析**：  
- IP 二进制：192.168.1.10000010
- 掩码二进制：255.255.255.10000000
- 网络地址：192.168.1.10000000 = 192.168.1.128

**记忆要点**：网络地址 = IP AND 子网掩码；广播地址 = 网络地址 OR (NOT 掩码)

## 高频考点 3：TCP 三次握手与四次挥手

**真题出现频率**：每年 1-2 题

### 典型题 3.1
**题目**：TCP 三次握手的第二个 SYN-ACK 包的作用是什么？
- A) 告诉客户端服务器已收到请求
- B) 告诉客户端服务器的初始序列号
- C) 建立连接
- D) A 和 B 都正确

**答案**：D  
**解析**：SYN-ACK 包既确认了客户端的 SYN，又发送了服务器的 SYN，实现了双向确认和序列号同步。

**记忆要点**：
- 第一次握手：客户端 SYN → 服务器
- 第二次握手：服务器 SYN-ACK → 客户端
- 第三次握手：客户端 ACK → 服务器
