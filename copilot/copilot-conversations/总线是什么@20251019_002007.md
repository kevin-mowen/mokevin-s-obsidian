### 总线系统（Bus System）

| 分类   | 内容                 |
| ---- | ------------------ |
| 按功能  | 数据总线、地址总线、控制总线     |
| 按范围  | 片内总线、系统总线、通信总线     |
| 性能指标 | 带宽、仲裁方式、同步方式、信号线数量 |

这些总线负责在计算机的不同组件之间传输数据和控制信息。例如，**数据总线**用于传输实际的数据；**地址总线**用于指定存储器的地址；而**控制总线**则用来发送控制信号。

### 总结

- **按功能分类**：根据其功能不同，可以分为数据总线、地址总线和控制总线。
- **按范围分类**：根据覆盖范围的不同，可以分为片内总线（在单个芯片内部）、系统总线（连接多个芯片）和通信总线（用于外部设备的通信）。
- **性能指标**：包括带宽（数据传输速率）、仲裁方式（如何决定哪个设备优先使用总线）、同步方式（时钟信号的使用情况）以及信号线的数量。
